# Ключевые особенности и преимущества

## Безопасность
- **Сквозное шифрование** - Сервер хранит только зашифрованные данные и не имеет доступа к секретам пользователя
- **Разделение ключей** - Разные криптографические ключи для локального хранилища, секретов и аутентификации на сервере
- **Современные алгоритмы** - Argon2id для генерации ключей, XChaCha20-Poly1305 для шифрования, BLAKE3 для хеширования
- **Детерминированная синхронизация** - Разрешение конфликтов через хеширование контента без расшифровки на сервере

## Фишки
- **SQLite в памяти с сериализацией** - Зашифрованное хранение данных без временных файлов
- **Эффективный протокол синхронизации** - Обнаружение изменений через хеширование минимизирует трафик
- **Чистая архитектура** - Интерфейс Cryptor предоставляет простой и безопасный API для всех криптоопераций
- **Минимальная поверхность атаки** - Сервер обрабатывает только аутентификацию и хранение зашифрованных данных

## Пользовательский опыт
- **Единственный мастер-пароль** - Единый доступ к локальному хранилищу и облачной синхронизации
- **Согласованность между устройствами** - Механизм синхронизации
- **Прозрачная безопасность** - Передовая криптография без сложностей для пользователя
- **Надежность** - Версионирование на основе хешей гарантирует целостность данных между устройствами
